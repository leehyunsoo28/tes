#프로필 #작업능력

가치관 및 신조

저는 목표를 달성하기 위해서는 철저한 계획과 실행이 반드시 필요하다고 믿습니다.
과거 목표만 세워두고 구체적인 실행 계획 없이 도전했을 떄, 노력에 비해 만족스러운 결과를 얻지 못한 경험이 있습니다.
그때부터 저는 막연한 열정보다 명확한 방향성과 단계적인 실행이 더 중욯다는 것을 깨닫게 되었습니다.
이후 자격증 취득을 준비하면서 세부 학습 계획을 수립하고 일정을 체계적으로 관리하였으며,
그 결과 목표한 자격증을 취득할 수 있었습니다.
또한 NCS 기반 퍼블리싱 전문 교육 과정에 참여하여 HTML, CSS, JavaScript 등 실무 역량을 체계적으로 쌓았습니다.
교육 과정 중 진행한 프로젝트에서는 개발 범위와 일정 계획을 먼저 정리한 뒤 작업을 수행하여 완성를 높이는 경험을 했습니다.
실제 기업의 개발 프로젝트는 명화한 기한과 요구 사항이 존재합니다.
저는 개발 고정과 작업 범위를 사전에 구체화하고, 계획을 실행으로 옭기는 태도를 통해 프로젝트의 품질을 높이는 개발자가 되겠습니다.

프로젝트 경험

NCS 기반 퍼블리싱 교육 과정에서 자동차 브랜드 웹사이트 리디자인 프로젝트를 수행하였습니다.
저는 BMW와 현대자동차 사이트를 선정하여 기존 기능은 유지하면서 사용자경험을 개선하는 방향으로 리디자인을 진행하였습니다.
CSS를 활용해 마우스 오버 시 색상과 배경이 자연스럽게 변경되도록 구현하고,
탭 메뉴를 추가하여 정보 접근성을 높였습니다.
또 한 JavaScript를 활용해 클릭 시 이미지 색상이 변경되거나 영상이 재생되는 기능을 구현하며 동적인 요소를 강화하였습니다.
프로젝트는 페이지 구성이 비교적 단순한 BMW 사이트부터 시작하여,
이후 현대자동차 아반떼 정보 사이트로 확장하였습니다.
특히 아반떼 사이트 제작 과정에서 JavaScript 오류가 반복적으로 발생하는 어려움을 겪었습니다.
이벤트 충돌과 변수선언 문제로 기능이 정상적으로 작동하지 않았지만,
오류 메시지를 분석하고 관련 자료를 찾아보며 원인을 파악했습니다.
해결이 어려운 부분은 지도 강사에게 질문하며 끝까지 수정해 나갔습니다.
이 경험을 통해 단순히 기능을 구현하는 것에 그치지 않고,
문제의 원인을 분석하고 해결하는 디버깅 역량의 중요성을 깨달았습니다.
또한 체계적인 코드 작성과 구조 설계가 오류를 줄이고 유지보수성을 높인다는 것을 배우며 한층 성장할 수 있었습니다.

성격의 장단점

저의 장점은 문제를 끝까지 해결하려는 끈기와 정해진 시간 안에 결과물을 완성하려는 태도입니다.
퍼블리싱 작업은 디자인 시안을 정확히 구현하는 것뿐 아니라,
일정 내에 완성도 있는 결과물을 만들어내는 과정이라고 생각합니다.
개인 프로젝트를 진행하며 작업 일정을 세분화해 계획을 수립했고,
중간 점검을 통해 수정 사항을 반영하며 일정에 맞춰 작업을 마무리했습니다.
또한 화면 구현 과정에서 예상치 못한 오류가 발생했을 때에도 임시 수정에 그치지 않고 원인을 분석하며 해결했습니다.
테스트를 반복해 문제를 개선했고, 일정에 차질이 없도록 관리했습니다.
다만, 오류 해결에 집중하다 보니 한 부분에 시간을 과도하게 투자하는 경우가 있었습니다.
이를 보완하기 위해 우선순위를 정하고 핵심 기능을 먼저 안정화한 뒤 세부 요소를 개선하는 방식으로 효율성을 높이고 있습니다.

지원동기 및 포부

웹퍼블리셔는 아이디어와 디자인을 실제 화면으로 구현하는 직무라는 점에서 큰 매력을느꼈습니다.
직접 마크업과 스타일을 설계하며 결과물이 즉각적으로 화면에 반영되는과정을 통해 구현의 보람을 경험했고,
작은 코드 수정이 사용자 경험에 영향을 준다는 점이 인상 깊었습니다.
특히 오류가 발생했을 때 원인을 분석하고 해결해 나가는 과정에서 문제 해결의 즐거움을 느꼈으며,
이를 통해 구조적으로 사고하는 습관을 기를 수 있었습니다.
또한 기업 사이트를 분석하고 기획하는 팀 프로젝트를 수행하며 협업의 중요성을 깊이 깨달았습니다.
각자의 역할이 유기적으로 연결되어야 완성도 높은 결과물이 나온다는 것을 경험했고,
가이드에 맞춘 작업과 원활한 소통이 프로젝트의 완성도를 높인다는 점을 배웠습니다.
입사 후에는 회사의 방향성과 프로젝트 목표를 정확히 이해하고,
웹표준과 접근성을 준수하는 퍼블리셔로 성장하고 싶습니다.
맡은 업무를 책임감 있게 수행하며,
팀과의 협업 속에서 안정적이고 완성도 높은 결과물을 만들어가는 인재가 되겠습니다.
